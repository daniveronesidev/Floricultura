# 🌸 Sistema de Floricultura

Sistema desktop desenvolvido em Java com o objetivo de gerenciar uma floricultura, permitindo o controle de produtos, estoque e vendas de forma simples e eficiente.

---

## 🚀 Funcionalidades

- Cadastro de produtos (flores e arranjos)
- Controle de estoque
- Registro de vendas
- Interface gráfica para interação com o usuário

---

## 🛠️ Tecnologias Utilizadas

- Java
- Java Swing
- Apache Ant
